# MoviNível 🚗

## 📖 Descrição
O **MoviNível** é um aplicativo de mobilidade urbana que será utilizado na Bolívia.  
Ele oferece as mesmas funcionalidades encontradas em apps tradicionais de mobilidade, porém com um diferencial: um sistema de níveis por indicação e cashback.  

O grande destaque do aplicativo é que tanto clientes quanto motoristas podem ganhar dinheiro ao divulgar o app, uma estratégia pensada para alavancar e escalar sua utilização.
 
---

## 🚀 Tecnologias Utilizadas
- **Kotlin** – Linguagem principal para desenvolvimento Android
- **Android Studio** – IDE utilizada
- **Firebase** – Autenticação e banco de dados em tempo real
- **Material Design** – Interface moderna e responsiva
- **Git/GitHub** – Controle de versão e colaboração
